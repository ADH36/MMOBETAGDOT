## Color Schemes

### Background Colors
- Main backgrounds: `Color(0.1, 0.1, 0.15, 1)` - Dark blue-gray

### Class Colors (Outfit)
- Warrior: `Color(0.8, 0.2, 0.2)` - Red
- Mage: `Color(0.3, 0.3, 0.9)` - Blue
- Rogue: `Color(0.3, 0.3, 0.3)` - Dark gray
- Archer: `Color(0.2, 0.7, 0.3)` - Green

### Skin Tones
- Light: `Color(1.0, 0.87, 0.74)`
- Medium: `Color(0.8, 0.6, 0.4)`
- Dark: `Color(0.6, 0.4, 0.2)`
- Darker: `Color(0.4, 0.3, 0.2)`

### Hair Colors
- Black: `Color(0.1, 0.1, 0.1)`
- Brown: `Color(0.4, 0.2, 0.1)`
- Blonde: `Color(0.8, 0.6, 0.2)`
- Red: `Color(0.8, 0.3, 0.1)`
- White: `Color(0.9, 0.9, 0.9)`
- Gray: `Color(0.5, 0.5, 0.5)`

## Screen Resolutions

Default window size: 1280x720
All UI elements are designed to scale with the window size using anchor presets.

## Accessibility Notes

- All buttons have clear text labels
- Character preview updates in real-time for immediate feedback
- Slider positions can be adjusted incrementally
- Back buttons available on all screens for easy navigation
- Disabled buttons are clearly indicated
